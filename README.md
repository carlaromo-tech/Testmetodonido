# Diagnóstico Familiar del Adolescente — Landing Page

Landing page estática para el **Diagnóstico Familiar del Adolescente**, proceso clínico de 3 sesiones diseñado por **Carla Romo Salcedo**, psicóloga clínica PUC, creadora del Método NIDO®.

Dirigida a madres profesionales con hijos adolescentes (12–18 años) que buscan entender qué está pasando en casa antes de tomar decisiones.

## Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Estructura semántica y copy completo |
| `styles.css` | Todos los estilos (mobile-first, CSS puro) |
| `script.js` | Animaciones GSAP + scroll suave Lenis |
| `assets/logo-nido.png` | **Agregar manualmente** — logo de Método NIDO® |

## Antes de publicar

1. **Agregar el logo**: coloca el archivo `logo-nido.png` dentro de la carpeta `/assets/`.
2. **Botones CTA**: busca `href="#test"` en `index.html` y reemplaza con la URL real del test.
